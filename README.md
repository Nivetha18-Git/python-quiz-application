# Python Quiz Application

## Overview

A desktop-based Quiz Application built using Python and Tkinter. The application allows users to log in, choose a difficulty level, take timed quizzes, and track their performance through score history.

## Features

- User Login and Session Management
- Easy, Medium, and Hard Difficulty Levels
- Timer-Based Quiz System
- Progress Tracking
- Previous and Next Navigation
- Skip Questions Option
- Automatic Score Calculation
- Quiz History Storage using JSON
- Interactive GUI using Tkinter

## Technologies Used

- Python
- Tkinter
- JSON
- Object-Oriented Programming (OOP)

## How to Run

1. Install Python 3.x
2. Download or clone this repository
3. Run the application

```bash
python quizapp.py
```
